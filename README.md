# prisma_orm

user authentication method using prisma, typescript and postgres

### Technologies Used:

- Typescrypt
- Node.js
- Express
- Bcrypt
- JWT
- Nodemon
- Postgres
- Prisma

### How To Run

To run you need to have docker and node.js installed and running on your machine.

```
docker-compose up
```

```
yarn prisma migrate dev --name init
```

```
yarn start
```
